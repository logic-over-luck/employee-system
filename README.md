# Employee Management System

Ein JavaFX-Projekt zur Verwaltung von Mitarbeitenden mit SQLite-Datenbank.  
Das Projekt dient als Lernprojekt im Bereich Java und fokussiert sich besonders auf Backend-Logik, Datenmodellierung, Validierung und Datenbankzugriff. Die aktuelle Version nutzt JavaFX für die Benutzeroberfläche, FXML für die Views sowie SQLite/JDBC für die Persistenz. :contentReference[oaicite:0]{index=0} :contentReference[oaicite:1]{index=1} :contentReference[oaicite:2]{index=2}

## Projektüberblick

In der Anwendung können Mitarbeitende angezeigt, angelegt, bearbeitet und gelöscht werden.  
Die Daten werden lokal in einer SQLite-Datenbank gespeichert. Außerdem gibt es eine Paginierung für die Tabellenansicht sowie eine Suchfunktion. :contentReference[oaicite:3]{index=3} :contentReference[oaicite:4]{index=4}

## Features

- Anzeige einer Mitarbeiterliste in einer JavaFX-Tabelle
- Detailansicht für einzelne Mitarbeitende
- Neue Mitarbeitende anlegen
- Bestehende Mitarbeitende bearbeiten
- Mitarbeitende löschen
- Speicherung der Daten in SQLite
- Paginierung der Listenansicht
- Suchfunktion
- Validierung zentraler Eingabedaten wie Name, E-Mail, IBAN, Gehalt und Adresse :contentReference[oaicite:5]{index=5} :contentReference[oaicite:6]{index=6}

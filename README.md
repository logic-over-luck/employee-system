# Employee Management System

Ein JavaFX-Projekt zur Verwaltung von Mitarbeitenden mit SQLite-Datenbank.  
Das Projekt dient als Lernprojekt im Bereich Java und fokussiert sich besonders auf Backend-Logik, Datenmodellierung, Validierung und Datenbankzugriff. Die aktuelle Version nutzt JavaFX für die Benutzeroberfläche, FXML für die Views sowie SQLite/JDBC für die Persistenz.

## Projektüberblick

In der Anwendung können Mitarbeitende angezeigt, angelegt, bearbeitet und gelöscht werden.  
Die Daten werden lokal in einer SQLite-Datenbank gespeichert. Außerdem gibt es eine Paginierung für die Tabellenansicht sowie eine Suchfunktion.

## Features

- Anzeige einer Mitarbeiterliste in einer JavaFX-Tabelle
- Detailansicht für einzelne Mitarbeitende
- Neue Mitarbeitende anlegen
- Bestehende Mitarbeitende bearbeiten
- Mitarbeitende löschen
- Speicherung der Daten in SQLite
- Paginierung der Listenansicht
- Suchfunktion
- Validierung zentraler Eingabedaten wie Name, E-Mail, IBAN, Gehalt und Adresse

## Tech Stack

- Java
- JavaFX
- FXML
- Maven
- SQLite
